This method is used to generate a random number between 0.0 and 1.0. The range is: 0.0<=Math.random<1.0. Different ranges
can be achieved using arithmetic.

                     static double random()
                     
EXAMPLE

                    public class Test{
                     public static void main(String args[]) { 
                     System.out.println(Math.random());
                     System.out.println(Math.random());
                        }
                    }
                    
RESULT

                   0.16763945061451657
                   0.400551253762343
